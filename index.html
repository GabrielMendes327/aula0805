    Algoritmo e pseudocódigos

    Algoritmo

  • É um conjunto de passos (chamados de comandos ou instruções) que devem ser seguidos para conseguir resolver um problema ou alcançar um determinado objetivo;
  • Não pode haver ambiguidade nos passos de um algoritmo.
  • Eles devem ser claros e precisos.
  • Um algoritmo pode ser representado em diferentes formas:
    1. Linguagem natural;
    2. Fluxograma;
    3. Pseudo-código;

    Pseudocódigos

  • Pseudocódigo é uma forma informal e genérica de representar um algoritmo, utilizando uma linguagem natural (como português ou inglês) para descrever os passos de um programa, sem a necessidade de uma sintaxe específica de linguagem de programação. Ele é usado como um plano ou guia para construir um programa, facilitando a compreensão e o desenvolvimento.

    • Estruturas de controle

    • Em determinadas situações é necessário executar ações de acordo com os dados fornecidos pelo programa. Em alguns casos, pode ser necessário que o programa execute uma determinada instrução repetidas vezes, por exemplo.
    • Sendo assim, controlar e manipular a ordem com que instruções serão executadas em função de dados fornecidos pelo programa é essencial, e é para isso que servem as estruturas de controle.

    • Linguagem de programação C

    Bibliotecas

  • Em C, as bibliotecas são conjuntos de funções predefinidas que fornecem funcionalidades comuns, como entrada/saída, manipulação de strings, funções matemáticas, etc.

    • Tipos de dados

  • char: Em C, o tipo char é usado para armazenar um único caractere, que é internamente representado como um inteiro, geralmente o código ASCII do caractere.
  • int: Em C, int é um tipo de dado que representa números inteiros (sem casas decimais), incluindo valores positivos, negativos e zero. É o tipo de dado inteiro padrão na linguagem C.
  • float: Em C, float é um tipo de dado que representa números de ponto flutuante, ou seja, números que podem ter casas decimais.
  • double: Em C,O tipo double é um tipo de ponto flutuante que é maior ou igual ao tipo float , mas é menor ou igual ao tamanho do tipo long double .
  • void: Em C, void é uma palavra-chave que indica que uma função não retorna valor ou que uma função não recebe argumentos.
