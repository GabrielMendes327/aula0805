Sobre a linguagem C

C é uma linguagem de programação de uso geral . Foi criada na década de 1970 por Dennis Ritchie e continua sendo amplamente utilizada e influente. Por design, os recursos do C refletem claramente as capacidades das CPUs visadas . Ele encontrou uso duradouro em código de sistemas operacionais, drivers de dispositivos e pilhas de protocolos , mas seu uso em software de aplicação vem diminuindo. C é comumente usado em arquiteturas de computadores que variam dos maiores supercomputadores aos menores microcontroladores e sistemas embarcados.

A linguagem C é considerada a linguagem mãe da programação. A linguagem C é uma linguagem de programação de baixo nível, procedural, estruturada e genérica, conhecida pela sua eficiência, portabilidade e controle de baixo nível sobre o hardware.
